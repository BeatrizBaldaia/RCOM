Parrot Security Repository

This server hosts a copy of the whole Parrot Security software archive.

We offer several mirror servers all around the world with the exact copy of this repository.
Choose the mirror server nearest to your location to improve download speeds.

Visit docs.parrotsec.org/mirrors-list for further info

If you use Parrot in your computer and you want to select one of these servers for your APT sources.list
use the
parrot-mirror-selector terminal command to change your apt sources settings
